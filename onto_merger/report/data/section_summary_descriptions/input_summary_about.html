The input consists of four tables. All tables are required to be present, however nodes_obsolete.csv can be an empty table with a header. For more details about the input table schemas and the required node ID schema, please read the docs.

All input tables are profiled, these details are presented in the Data Profiling section.

The pipeline validates all inputs and halts the process if any errors are found. Validation details are presented in the Data Tests section.
