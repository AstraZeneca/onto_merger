This table provides links to the main folders used or produced by the pipeline.

  • The Input data folder contains pipeline input and configuration.
  • The Domain ontology folder contains the result of the pipeline process.
  • The Analysis data & plots folders provide the content of this report and some helpful files for debugging the process.
  • The Data profiling reports folder contains the HTML report files produced by Pandas profiling.
  • The Data test configurations & report folders contain the validation setting JSON-s and the resulting HTML report.
  • The Dropped mappings folder contain the mapping that could not be used during the alignment process (due to multiplicity issues).
  • The Pipeline logs folder contain the log files produced by OntoMerger.

For more details about the project folder organisation and content, please read the docs.
